# Python-number-guessing-game
A simple number guessing game built with Python

## Description
The program generates a random number between 1 and 10, and the player tries to guess it. 
After each guess, the program provides hints whether the guess is too high or too low. 
Once the correct number is guessed, a congratulatory message is displayed.

## Features
- Random number generation using Python's `random` module
- User input handling
- Loops and conditional statements
- Basic interactive game logic

## Skills Demonstrated
- Python basics: variables, loops, conditionals
- Simple user interface through terminal input/output
- Problem-solving logic in game design
